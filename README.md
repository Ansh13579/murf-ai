## ✨ Features

### 📄 File → Speech
Upload documents in **any format** and get translated text + narrated audio:
- **Supported formats**: TXT, PDF, DOCX, CSV, MD, JSON, XML
- Auto-chunks large text and merges MP3 fragments in order
- Translates up to 50 lines per file with Murf Translation API
- Line order and structure is preserved

### 💬 AI Chat + Voice
Chat with an **AI assistant** powered by Groq (Llama 3.1) — every reply is:
- Generated by AI (not just translated)
- Translated into your target language via Murf
- Spoken aloud with Murf TTS voices

## 🏗️ Architecture

```
Browser (HTML/CSS/JS)
        │
        ├─ POST /api/translate-file   →  Upload file → Extract text → Murf Translate → Murf TTS → Merged MP3
        ├─ POST /api/translate-chat    →  User message → Groq AI reply → Murf Translate → Murf TTS
        ├─ GET  /api/voices            →  Available languages & voices
        └─ GET  /api/health            →  Server status
        │
   Flask Backend (server.py)
        │
        ├── Groq API (Llama 3.1 8B)   →  AI chatbot brain
        ├── Murf Translation API       →  Text translation
        ├── Murf TTS API               →  Speech synthesis
        └── PyDub + FFmpeg             →  Audio merging
```

## 🛠️ Run Locally

```bash
# Clone
git clone https://github.com/Ansh13579/murf-ai.git
cd murf-ai

# Install dependencies
pip install -r requirements.txt

# Set environment variables
cp .env.example .env
# Edit .env with your API keys (see below)

# Run
python server.py
# → Open http://localhost:5000
```

### Required API Keys

| Key | Where to get it | Free tier |
|-----|-----------------|-----------|
| `MURF_KEY` | [murf.ai](https://murf.ai) → Dashboard → API Keys | Limited |
| `GROQ_KEY` | [console.groq.com](https://console.groq.com) → API Keys | 30 req/min, 14,400/day |
| `GEMINI_KEY` *(optional fallback)* | [ai.google.dev](https://ai.google.dev) | 15 req/min |

## 🐳 Deploy with Docker

```bash
docker build -t murf-ai .
docker run -p 5000:5000 \
  -e MURF_KEY=your_key \
  -e GROQ_KEY=your_key \
  murf-ai
```

## 📝 API Limits (enforced in code)

| Limit | Value |
|-------|-------|
| Translation batch | ≤ 10 sentences per request |
| Translation chars | ≤ 4,000 chars per sentence |
| TTS chars | ≤ 3,000 chars per request (auto-chunked) |
| File lines | ≤ 50 lines (to prevent timeouts) |
